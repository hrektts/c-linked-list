pub struct CLinkedList<T, P, F: Fn(&T) -> P> { /* fields omitted */ }

A linked list for wrapping a C linked list.

This struct is created by the from_const_ptr method and from_mut_ptr method from a C linked list that elements are linked by immutable pointers and mutable pointers, respectively. See their documentation for more.
